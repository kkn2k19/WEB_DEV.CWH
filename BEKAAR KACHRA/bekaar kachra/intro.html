What is HTML?

1. It stands for Hyper text markup language.

2. It is the standard markup language for creating web pages.

3. It describes the structure of a web page.

4. It consists of a series of elements.

5. It's elements tell the browser how to display the content.

6. It's elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
